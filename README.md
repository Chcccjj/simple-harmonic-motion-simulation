# Simple Harmonic Motion Laboratory

An interactive physics simulation platform for exploring Simple Harmonic Motion (SHM), designed for AP Physics and introductory university mechanics.

Developed by Chris Chen.

---

## Overview

This project provides a virtual laboratory environment for studying two classical SHM systems:

- Simple Pendulum
- Mass-Spring Oscillator

Users can modify physical parameters in real time and observe how motion, energy, phase space trajectories, and oscillation periods change under different conditions.

The simulator is intended for:

- AP Physics 1
- Introductory Mechanics
- Classroom Demonstrations
- Independent Physics Exploration

## Physics Principles

### Simple Pendulum

The theoretical period is

T = 2π√(L/g)

where:

- T = period
- L = pendulum length
- g = gravitational acceleration

For small angles, the period is independent of mass.

---

### Mass-Spring Oscillator

The theoretical period is

T = 2π√(m/k)

where:

- m = mass
- k = spring constant

The period is independent of oscillation amplitude.
